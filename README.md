# PHPA.D.Conversion
西暦から和暦を割り出す

## 処理
昭和元年から10年までの西暦和暦変換を行います

## コード
```
<?php
for ($seireki=1926; $seireki < 1936; $seireki++) {
    echo "西暦".$seireki."年は、";
    $syowa = $seireki - 1925;
    echo "昭和".$syowa."年です。\n";
}
```

## 出力結果  
```
西暦1926年は、昭和1年です。
西暦1927年は、昭和2年です。
西暦1928年は、昭和3年です。
西暦1929年は、昭和4年です。
西暦1930年は、昭和5年です。
西暦1931年は、昭和6年です。
西暦1932年は、昭和7年です。
西暦1933年は、昭和8年です。
西暦1934年は、昭和9年です。
西暦1935年は、昭和10年です。
```
